About

I am a passionate researcher at the intersection of AI, security, and distributed systems. My work focuses on translating cutting-edge theory into practical solutions.

Currently, I specialize in Federated Learning, creating privacy-preserving models for environmental and healthcare applications. Simultaneously, I am developing frameworks for Smart Contract Security using multimodal AI to detect vulnerabilities in blockchain systems.

Research Highlights

Privacy-Preserving Environmental Monitoring

Developed a federated learning framework for deforestation detection using satellite imagery. Achieved 0.85 IoU while maintaining strict data privacy across distributed nodes.

VulnFusion & QuadraCode AI

Created multimodal AI frameworks for smart contract vulnerability detection, combining code analysis with graph neural networks for enhanced security auditing.

Big Data Systems Optimization

Conducted comparative analysis of Apache Spark vs Apache Airflow for large-scale data processing to provide performance benchmarks for enterprise ML deployments.
